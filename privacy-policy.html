Information We Collect

This educational application may collect the following types of information to demonstrate data handling practices:

Profile Information

  • Name and email address from social login
  • Profile picture (if provided by OAuth)
  • Basic demographic information

Health Data

  • Menstrual cycle tracking data
  • Symptom logging information
  • Personal health notes and observations

Usage Analytics

  • App usage patterns and preferences
  • Feature interaction data
  • Performance and error logging

How We Use Your Information

The information collected is used solely for educational and functional purposes:

  • Providing period tracking and prediction functionality
  • Maintaining your user account and personalized experience
  • Displaying your personal cycle data and health insights
  • Demonstrating secure data handling practices
  • Learning about user experience design and data visualization

Data Storage and Security

Security Commitment: As an educational project focused on best practices, we implement industry-standard security measures to protect your data.

Security Measures

  • Data encryption both in transit and at rest
  • Secure database storage with access controls
  • Regular security audits and code reviews
  • Implementation of OWASP security guidelines

Educational Context

  • Data is stored for the duration of the academic project
  • No commercial use or monetization of data
  • Data access limited to project team and instructors
  • Regular security practices demonstrations

Third-Party Authentication

This application uses OAuth 2.0 for secure authentication with trusted providers:

Facebook Login

We receive basic profile information as permitted by Facebook's API. We do not store your Facebook password or access your private Facebook data.

Google OAuth

We access only the basic profile information you consent to share. We do not store your Google password or access your Gmail or other Google services.

Data Retention

As an educational project, our data retention policy is designed with learning objectives in mind:

  • User data retained for the duration of the academic term
  • Data may be anonymized for educational analysis
  • All personal data securely deleted at project conclusion
  • Users can request immediate data deletion at any time

Your Rights

You have comprehensive rights regarding your personal data:

Access Rights

  • Download a copy of your data
  • Understand how your data is being used

Control Rights

  • Correct any inaccurate information
  • Update your preferences at any time
  • Restrict certain data processing

Deletion Rights

  • Request complete account deletion
  • Remove specific data entries
  • Withdraw consent at any time
